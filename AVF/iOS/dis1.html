    3. Briefly describe some past 'lessons learned' regarding workflow as it relates to research, design, development, etc.

    Taking the research and workflow one step at a time is the biggest lesson that I have learned. For example, I think creating the HTML first and then adding css, and lastly adding javascript can help manage the workflow. Jumping around from one file to the other right from the start can get tricky and I start to think to much. Have the HTML solidified (more or less) makes working on everything else that much easier. Adding in pictures and icons goes hand in hand with css. In terms of research, I think that reading references and guides in full is important. I've noticed many times that I felt comfortable implementing a new plugin or jQuery Mobile functionality to only find out that I was not able to fully make everything work. I would go back to the reference and continue to read through the examples and documentation to find why I was having issues and how to correct them. If I had read through everything fully the first time I might not have run into the same issues.
